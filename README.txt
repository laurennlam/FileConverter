# FileConverter
Convert some files into an other type.

## Executable folder
This folder contain the last executable version.
It can be used without any dll.

## How to use executable
Converter.exe --input pathToYourInputFile --output pathToYourConvertedFile
The extension of the paths will define the conversion.

## Supported formats
The supported format are the followings:
- obj
- ply
- stl
- vtk
- vtp
